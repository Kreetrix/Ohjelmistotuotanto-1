# User Stories - Card Memo 

## User story 1 : User Registration
**As a** new user  
**I want to** create an account with username, email, and password  
**So that** I can access the flashcard application 

## User story 2 : User Login
**As a** registered user  
**I want to** log in with my username and password  
**So that** I can access my personalized flashcard decks

## User story 3 : Create Deck
**As a** user  
**I want to** create a new flashcard deck with name and description  
**So that** I can organize my study materials

## User story 4 : Edit Deck
**As a** deck owner  
**I want to** edit my deck's name, description, and visibility  
**So that** I can update deck information as needed

## User story 5 : Delete Deck
**As a** teacher or admin  
**I want to** delete decks  
**So that** I can remove outdated or incorrect content

## User story 6 : Create Card
**As a** user  
**I want to** create flashcards with question and answer  
**So that** I can build my study materials

## User story 7  : Edit Card
**As a** card creator  
**I want to** edit my flashcard content  
**So that** I can correct mistakes or improve clarity

## User story 8 : Filter Cards by Deck
**As a** user  
**I want to** filter cards by deck  
**So that** I can view cards from specific decks only

## User story 9  : Start Study Session
**As a** student  
**I want to** start a study session with a deck  
**So that** I can practice and memorize the content

## User story 10 : Flip Card and Rate Knowledge
**As a** student  
**I want to** flip cards to see the answer and indicate if I knew it  
**So that** I can test my knowledge and track my progress

## User story 11 : Complete Study Session
**As a** student  
**I want to** see my results after completing all cards  
**So that** I can track my performance

## User story 12 : Switch Language
**As a** user  
**I want to** change the application language  
**So that** I can use the app in my preferred language

## User story 13 : View Translated Content
**As a** multilingual user  
**I want to** see deck names and card content in my selected language  
**So that** I can understand and study content in my preferred language

## User story 14 : Student Permissions
**As a** student  
**I want to** manage only my own decks and cards  
**So that** I have appropriate access to content

## User story 15 : Teacher/Admin Permissions
**As a** teacher or admin  
**I want to** manage all decks and cards  
**So that** I can curate content for all users
